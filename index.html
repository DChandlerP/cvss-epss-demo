Vulnerability Lookup

What is CVSS?

The Common Vulnerability Scoring System (CVSS) is a framework for assessing the severity of security vulnerabilities. It provides a score from 0 to 10, with a higher score indicating a more severe vulnerability. CVSS considers factors such as access complexity, exploitability, and impact. For more information, visit the CVSS official page.

What is EPSS?

The Exploit Prediction Scoring System (EPSS) estimates the probability of a vulnerability being exploited. It provides a score between 0 and 1, indicating the likelihood of exploitation within 30 days. EPSS complements CVSS by focusing on the likelihood of exploitation rather than just severity. Learn more at the EPSS official page.
